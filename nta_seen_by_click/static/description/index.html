Seen by Click

Manage every click on any record within Odoo.

Manage the general purpose of storing the history

Store when, who and what.

Store every click history of any record.

Know who and when clicked on any record for the first time.

Use the history to create cool features like changing the color of a record in the messaging.

Easy to use

No dependencies required.

Just install it and use it by inheritance.

API

A short document to give you a hint to how to use it.

you can inherit the model "nta.seen.by.click" in your model and use it by the function field "nta_special_seen" on any instance of your model to see if the current user had seen the current record.
